# Enterprise Rules Engine

An enterprise rules engine for all your needs


## Installation

Add this line to your application's Gemfile:

    gem 'rules_engine'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rules_engine

## Usage

Require everything

    ```Ruby
    require 'rules_engine'
    RulesEngine.new
    ```
